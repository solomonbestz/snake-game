# SNAKE GAME PYGAME 

A simple python snake game made with pygame 2d game library.

## PREREQUISITE

- **Python**
- **Pygame 2d**

## Getting Started

### Installation

1. Clone the repository

    ```bash
        git clone https://github.com/solomonbestz/snake-game.git
    ```

2. Install Library

    ```bash
        pip install -r requirements.txt
    ```

### Usage

1. Run Game

    ```bash
        py main.py
    ```
